Task 1: Basic Network Sniffer
Developed as part of Cyber Security Internship at Rhombix Technologies.
Description:
A python script using the Scapy library to capture live network packets and display source/ Destination IPs and protocols (TCP/UDP).
How to use:
1. Install Scapy:pip install scapy
2. Run sniffer.py as Administrator
